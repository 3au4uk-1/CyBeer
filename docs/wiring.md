# CyBeer — распиновка (кратко)

Значения из `firmware/components/cybeer_config/include/cybeer_config.h`.

| Функция        | GPIO |
|----------------|------|
| Switch         | 9    |
| TM1637 CLK     | 6    |
| TM1637 DIO     | 7    |
| WS2812 data    | 8    |
| Battery ADC    | 0    |

**Замечания**

- Линия WS2812 — через RMT; при 5 V ленте и 3.3 V MCU может понадобиться согласование уровня.
- Переключатель — по схеме проекта (подтяжка, фильтр дребезга в `cybeer_switch`).
- АЦП батареи — калибровка/делитель по вашей плате.
